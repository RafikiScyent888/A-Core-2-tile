# A+ Core 2 Tile

A standalone landing page for CompTIA A+ Core 2 (220-1202) resources — part
of the **Cyber Warrior Program**. Royal Blue background matching the
[Cyber Warrior Command Center](https://rafikiscyent888.github.io/Cyber-Warrior-Command-Center/),
with a single Royal Purple "Core 2" tile linking out to:

- [Core 2 Quizzes](https://rafikiscyent888.github.io/Core-2-quizzes/)
- [Core 2 Sims](https://rafikiscyent888.github.io/Core-2-Sims/)
- [A+ Acronyms](https://rafikiscyent888.github.io/A-acronym-games./)

Plain HTML/CSS, no build step, no dependencies. Mobile-friendly.

## Hosting

GitHub Pages, deployed from `main` / root — Settings → Pages → Source:
Deploy from a branch → `main` → `/ (root)`.
